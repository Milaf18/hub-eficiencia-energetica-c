# ⚡ Hub de Eficiência Energética para IoT

Este projeto é um simulador de monitoramento inteligente para laboratórios de informática, desenvolvido em **Linguagem C**. O foco principal é a **Computação Sustentável (Green AI)**, reduzindo o desperdício de energia através de políticas automáticas de hibernação.

## 🌿 Foco em Sustentabilidade
O sistema foi projetado para combater o desperdício energético, focando em:
- **Energy Efficiency:** Redução ativa do consumo em Watts.
- **Carbon Footprint:** Minimização da pegada de carbono operacional.
- **Sustainable Computing:** Automação baseada em inatividade.

## 🚀 Funcionalidades
*   **Monitoramento Automático:** Hibernação após 20 min de inatividade.
*   **Gestão de Prioridades:** Máquinas críticas (servidores) não são afetadas pela hibernação automática.
*   **Painel de Controle:** Visualização em tempo real do estado das máquinas (Ligada, Hibernando, Desligada).
*   **Relatórios de Economia:** Cálculo exato de Watts economizados durante a simulação.

## 🛠️ Tecnologias
- Linguagem C
- Bibliotecas: `stdio.h`, `stdlib.h`, `string.h`, `time.h`
